## Development Logging Module
### Overview

This module provides additional logging utilities intended for development and debugging purposes only.

It helps developers capture detailed runtime information such as API calls, payloads, responses, and internal processing steps. The logs generated by this module can assist in troubleshooting issues during development and testing.

⚠️ Important:
This module is not designed for production environments. It may log sensitive data and generate high log volumes, which can impact performance and security.

Purpose

The module was created to:

- Provide detailed debug logs during development
- Capture request/response payloads
- Trace internal application flows
- Simplify debugging during feature implementation
- Assist in identifying integration or data processing issues
